🎥 第三步：怎么做一条视频

全程只需要 Seedance 2.0 + 剪映加字幕，40分钟一条

1
选一个场景
从上面5类里选一个具体画面，比如"深夜被窝刷手机看到新物料"
2
写Prompt（提示词）
告诉AI你要什么画面。按下面的模板填空就行👇
✍️ Prompt万能模板
[场景] 在哪里？什么时间？什么环境？ [人物] 中国年轻女生，穿什么，什么状态 [动作] 她做了什么？表情怎么变化？ [镜头] 特写/中景/远景，固定/缓慢推进 [风格] 电影质感，竖屏9:16
3
投入Seedance 2.0生成
把写好的Prompt粘贴进去，生成15秒视频。不满意就微调Prompt重新生成
4
加字幕发布
用剪映给视频加4-6句字幕（可以加简单淡入动效），然后发小红书

✅ 3个现成Prompt，直接复制用

示例① 被窝刷手机
[场景] 夜晚，中国年轻女生躺在床上，被子拉到下巴，脸被手机柔和的蓝色光芒照亮。 [动作] 滑动手机浏览，突然停下，眼睛睁大，忍不住露出害羞的笑容，咬下唇，兴奋地把被子蒙过头。 [镜头] 特写，固定机位，她笑的时候缓慢推进。 [风格] 电影质感，暖色调，胶片颗粒感，温馨氛围，竖屏9:16。

配字幕：深夜刷到他的新物料 / 控制表情失败 / 被窝里的追星人 / 你是不是也这样

示例② 演唱会散场
[场景] 演唱会结束后的城市夜晚街道，中国年轻女生独自走在路上，手里拿着微微发光的荧光棒，穿着粉丝应援卫衣。 [动作] 缓慢走着，抬头看天空，深呼吸，嘴角浮现温柔的微笑。戴着耳机，正在听歌。 [镜头] 中景跟拍，从背后缓慢弧线转到侧脸。街灯虚化光斑。 [风格] 电影质感，冷色调搭配暖黄路灯点缀，胶片颗粒感，竖屏9:16。

配字幕：散场了 / 但耳机里还在单曲循环 / 路灯下的满足感 / 下次见

示例③ 上班摸鱼
[场景] 中国年轻女生坐在办公室工位前，电脑屏幕显示着工作文档，偷偷把手机藏在桌面下。 [动作] 紧张地左右张望，低头看手机，突然绽放灿烂笑容。同事经过，她瞬间坐直假装打字。 [镜头] 中近景，略微倾斜角度，办公室自然光线。 [风格] 写实风格，自然光，微妙幽默感，生活切片，竖屏9:16。

配字幕：工位上收到他的新消息 / 表情管理失败 / 追星人的职场生存指南

💬 Prompt写得好和差区别有多大？
因为我们不做复杂剪辑，Seedance直出就是成片，所以Prompt质量=成片质量。写的时候注意：
• 要具体——"笑"不如"忍不住露出害羞的笑容，咬下唇"
• 写动作变化——15秒内要有一个情绪转折（平静→激动/搞笑→感动）
• 别忘镜头和风格——"电影质感"和"竖屏9:16"必写
• 绝对不提任何真实明星姓名——只描述粉丝角色

🔥 第四步：看到热点怎么变成视频

这是最核心的技能——任何热点都能30分钟变成一条视频脚本

🧠 记住这个口诀就行：
热点 → 情绪 → 粉丝什么时候也有这种情绪 → 拍那个场景

👇 拆解成3步：

1
看到热点，先想：这件事让人什么感觉？
感动？搞笑？骄傲？心疼？思念？——提取一个核心情绪
2
再想：粉丝追星时，什么场景也会有这种情绪？
把热点的情绪"翻译"成一个粉丝追星的画面
3
写Prompt + 字幕，生成视频
用前面的模板填好，投入Seedance生成

💬 追热点的速度要多快？
• 王一博相关热点（官宣/物料/活动）→ 当天4小时内发，抢第一波
• 可借势的娱乐/社会热点 → 当天或次日发
• 节日/季节（七夕/高考/开学/过年）→ 提前3-7天准备好
• 天气/日常（下雨/降温/周一）→ 当天随手发

热点去哪看？王一博超话、小红书热搜、微博热搜、抖音热榜。每天花5分钟扫一眼就行。

💬 为什么标题这么重要？
小红书用户刷信息流时，决定是否点开只需要1秒。标题要在1秒内让粉丝觉得"这说的是我"或"这我必须看"。所以群体认同型标题效果最好——直接喊话"你是不是也这样"。

前30天怎么安排

第1周（试水）：发1条图文自我介绍 + 4条视频，测试5种不同场景，看哪个数据最好

第2周（加速）：围绕第1周数据最好的类型多发，开始追热点

第3周（冲量）：启动一个系列（如"追星人的24小时"），发互动征集帖

第4周（稳定）：固定每周栏目，做一次数据复盘，调整方向

💬 为什么不建议日更？
小红书算法不像抖音那样重视日更频率。1条高质量视频的效果 > 3条凑数的。而且每周4-5条的节奏你不会太累，能坚持得更久。刚开始最怕的不是发得少，是做两周觉得累就放弃了。

💬 怎么判断数据好不好？
主要看这几个数：
• 互动率≥5%（点赞+评论+收藏 ÷ 浏览数）——说明内容有共鸣
• 收藏率≥6%——说明内容有价值，用户想留着看
• 评论里有"这不就是我"类内容——方向对了
• 每月出1-2条过万播放——爆款在路上了

数据不好不要慌，看看哪条最好、哪条最差，找区别，调整就行。

🛡️ 红线（必须遵守）

  • 不出现王一博面容——偶像只存在于字幕文案中

  • 每篇加 #AI创作 标签——标明是AI生成内容
  • 不冒充官方——简介/内容里标"粉丝向创作"
  • 不造谣不传未证实信息——只做情感共鸣内容
  • 不引战不拉踩——不碰其他艺人粉圈
  • BGM用无版权音乐——用平台内置或免版权曲库
